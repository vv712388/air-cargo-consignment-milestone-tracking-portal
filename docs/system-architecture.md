# System Architecture

## Air Cargo Consignment Milestone Tracking Portal

### Components

### 1. Frontend Layer

* HTML
* CSS
* JavaScript
* Shipment Booking Form
* Dashboard Summary
* Shipment Records Table

### 2. Backend Layer

* Node.js
* Express.js
* REST APIs

### 3. Database Layer

* Customer Information
* Shipment Information
* Milestone Status Records

### 4. Tracking Module

* Booking Confirmation
* Documentation Verification
* Cargo Acceptance
* Departure Tracking
* Transit Tracking
* Arrival Tracking
* Customs Clearance
* Delivery Confirmation

### 5. Reporting Module

* Dashboard Statistics
* Shipment Reports
* Delivery Reports

### Architecture Flow

Customer → Frontend → Backend API → Database

Database → Backend API → Frontend Dashboard

Operations Staff → Backend API → Shipment Status Updates

Admin → Dashboard → Reports & Monitoring
