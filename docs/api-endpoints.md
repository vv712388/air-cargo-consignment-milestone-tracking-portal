# API Endpoints

## 1. Create Shipment

**Method:** POST

**Endpoint:**
/shipments

**Purpose:**
Create a new shipment record.

---

## 2. Get All Shipments

**Method:** GET

**Endpoint:**
/shipments

**Purpose:**
Retrieve all shipment records.

---

## 3. Get Shipment Details

**Method:** GET

**Endpoint:**
/shipments/:id

**Purpose:**
Retrieve details of a specific shipment.

---

## 4. Update Shipment Status

**Method:** PUT

**Endpoint:**
/shipments/:id/status

**Purpose:**
Update shipment milestone status.

---

## 5. Dashboard Summary

**Method:** GET

**Endpoint:**
/dashboard

**Purpose:**
Retrieve dashboard statistics and shipment summaries.
